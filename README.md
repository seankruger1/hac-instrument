# HAC Readiness Instrument v5.3

**Human–AI Collaboration Readiness Instrument**  
Dr Sean Kruger · Dept of Informatics, EMS · University of Pretoria

## About

This instrument measures readiness for responsible Human–AI Collaboration across four UP populations:
- Undergraduate Students
- Postgraduate Students (Honours/Masters/PhD)
- Academic Staff
- Professional Staff

## Features

- 41-item Likert battery (7 constructs: DAL, ESI, CT, GAD, IEC, AAQ, FOWR)
- Adaptive scenario routing based on population (DEM0) and confidence tier
- 32 UP-grounded scenario cases (8 per population)
- Personalised readiness report with focus area recommendations
- Email report to participant

## Running Locally

```bash
npm install
npm run dev
```
